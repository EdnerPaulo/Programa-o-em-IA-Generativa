🧩 5 - Framework de Prompt (o coração do projeto)
📌 CRAFT
Context: A qualidade da explicação depende do prompt.

Role: Engenheiro de Prompt.

Action: Criar um prompt padrão eficiente.

Format: Template reutilizável.

Target: Sistema de IA.
🧠 Prompt base (ESSENCIAL)
Você é um professor de programação para iniciantes.

Explique o código abaixo linha por linha em português simples.

Regras:
- Use linguagem fácil
- Explique como se fosse para alguém iniciante
- Dê exemplos quando necessário
- Não use termos técnicos difíceis

Código:
{codigo}

👉 Isso é o seu diferencial 🔥

🖥️ 6 - Interface (Streamlit)
📌 CRAFT
Context: Usuário precisa interagir facilmente com a IA.

Role: Desenvolvedor Front-end Python.

Action: Criar interface simples no Streamlit.

Format: Estrutura de tela.

Target: Usuários iniciantes.
🎯 Layout
Campo para colar código
Botão: "Explicar código"
Área de resposta
Opção:
nível: básico / intermediário
💻 Exemplo simples
import streamlit as st

st.title("IA Explica Código")

codigo = st.text_area("Cole seu código aqui")

if st.button("Explicar"):
    explicacao = gerar_explicacao(codigo)
    st.write(explicacao)
