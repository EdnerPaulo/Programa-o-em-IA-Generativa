## **Context**
Plataforma educacional de programação voltada para iniciantes brasileiros que enfrentam dificuldades em entender códigos complexos, documentação técnica em inglês e jargões de programação.

---

## **Role**
Engenheiro de Machine Learning e Desenvolvedor Full Stack especializado em NLP (Processamento de Linguagem Natural) e criação de ferramentas educacionais acessíveis.

---

## **Action**
Desenvolva uma solução completa seguindo estas 7 etapas:

### **1. Problema**
- Iniciantes desistem de programar por não entenderem código
- Documentação técnica é intimidadora e em inglês
- Falta de ferramentas que "traduzam" código para linguagem simples
- Barreira do idioma para brasileiros (maioria do conteúdo em inglês)

### **2. Modelo TensorFlow - Solução Técnica**
```python
# Estrutura do modelo de explicação de código
modelo = Sequential([
    # Camada de embedding para tokens de código
    Embedding(vocab_size=50000, embedding_dim=256),
    
    # LSTM bidirecional para entender contexto do código
    Bidirectional(LSTM(256, return_sequences=True)),
    
    # Attention mechanism para focar em partes relevantes
    AttentionLayer(),
    
    # Decoder para gerar explicações em português
    Dense(512, activation='relu'),
    Dropout(0.3),
    Dense(vocab_size_pt, activation='softmax')
])

# Treinamento com dataset pareado: código → explicação
modelo.compile(
    optimizer='adam',
    loss='sparse_categorical_crossentropy',
    metrics=['accuracy', 'bleu_score']  # BLEU para qualidade da tradução
)
```

**Dataset de treino:**
- 100k+ pares (código Python, explicação em português)
- Curadoria de tutoriais brasileiros, fóruns, documentação traduzida
- Data augmentation com variações de código

- **Containerização:** Docker + docker-compose

### **4. APIs de IA - Comparativo e Escolha**

| API | Vantagem | Uso no Projeto |

| **Gemini 1.5 Pro** | Melhor custo-benefício, contexto 1M tokens | Explicação de códigos longos |

**Estratégia híbrida:** Modelo TensorFlow próprio para 80% dos casos + APIs para casos complexos (fallback).

### **6. Streamlit - Interface Gráfica**

```python
import streamlit as st
import tensorflow as tf
from code_explainer import explain_code

# Configuração da página
st.set_page_config(
    page_title="Códego - Seu Código Explicado 🇧🇷",
    page_icon="💡",
    layout="wide"
)

# Sidebar com configurações
with st.sidebar:
    st.title("⚙️ Configurações")
    nivel = st.select_slider(
        "Seu nível:",
        options=["🐣 Iniciante Total", "🐥 Iniciante", "🐔 Intermediário"]
    )
    linguagem = st.selectbox(
        "Linguagem do código:",
        ["Python", "JavaScript", "Java", "C++", "Outra"]
    )

# Área principal
col1, col2 = st.columns(2)

with col1:
    st.header("📝 Cole seu código aqui")
    codigo = st.text_area(
        "Código:",
        height=400,
        placeholder="Cole seu código aqui...\n\nExemplo:\nprint('Olá, Mundo!')\nfor i in range(5):\n    print(i)"
    )
    
    if st.button("🔍 Explicar Código", type="primary"):
        with st.spinner("Analisando seu código... 🤔"):
            explicacao = explain_code(codigo, nivel, linguagem)
            st.session_state['explicacao'] = explicacao

with col2:
    st.header("🎓 Explicação")
    if 'explicacao' in st.session_state:
        st.markdown(st.session_state['explicacao'])
        
        # Features extras
        with st.expander("🎯 Dicas de Melhoria"):
            st.write("Sugestões para otimizar seu código...")
        
        with st.expander("📚 Conceitos Relacionados"):
            st.write("Links e materiais complementares...")

# Footer
st.markdown("---")
st.markdown("💡 **Dica:** Quanto mais específico seu código, melhor a explicação!")
```
